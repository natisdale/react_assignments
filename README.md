## Exercise & Assignment Code

To launch one of the exercises or assignments locally:

1. Navigate to the target subfolder using `cd week<n>\exercise` or `cd week<n>\assignment`
1. Run `npm install` to install required libraries
1. Run `npm run dev` to launch dev web server
1. Open the link returned by the above step
